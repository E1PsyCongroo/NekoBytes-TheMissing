## Week3

  - 指针
- C 程序的硬件视角，继续理解**抽象**:
  - 变量, 指针, 数组, 字符串, 函数, 代码在内存中存储

  - 操作系统提供系统库的 ABI
- 提供一些 C 语言自学指引材料
- 参考资料:
